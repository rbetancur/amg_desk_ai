# Documentación de Seguridad

Este directorio contiene documentación relacionada con auditorías de seguridad, políticas y reportes de seguridad del proyecto.

## Contenido

- **SECURITY_AUDIT_REPORT.md**: Reporte de auditoría de credenciales hardcodeadas y validación de seguridad del proyecto.

## Propósito

Este directorio separa la documentación de seguridad y auditorías de la documentación general del proyecto, facilitando su mantenimiento y acceso.

## Notas

- Los reportes de auditoría se generan periódicamente para validar la seguridad del código.
- Esta documentación es de referencia interna y no debe contener información sensible.
